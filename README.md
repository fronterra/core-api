# Fronterra Open Source API

## Introduction
The Fronterra API is an open source pollution reporting API. The core api provides CRUD operations for two enpoints: `reports`, and  `admin`.

## Endpoints
### 1. `/reports`


**GET**
- header
- url params
- query string params
- body

**POST**
- header
- url params
- query string params
- body

**DELETE**
- header
- url params
- query string params
- body

### 2. `/admin`


**GET**
- header
    - JWT token
- url params
- query string params
- body

**POST**
- header
- url params
- query string params
- body

**PATCH**
- header
    - JWT token
- url params
- query string params
- body


**DELETE**
- header
    - JWT token
- url params
- query string params
- body
